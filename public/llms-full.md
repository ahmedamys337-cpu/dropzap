## What Is DropZap?

DropZap is a free browser-based media downloader. Users paste a URL from
Instagram, TikTok, Twitter/X, Facebook, Reddit, Pinterest, or Threads into
DropZap and the file downloads directly to their device in HD quality.
No software installation, no account creation, and no fees.
DropZap is powered by yt-dlp, a trusted open-source media extraction library.

## How DropZap Works

1. Copy the URL of the video or post from the platform.
2. Go to dropzap.digital and select the correct tool tab.
3. Paste the URL into the input field.
4. Click Download.
5. The file downloads directly to your device's downloads folder.

Processing time: 3-5 seconds for most content.

## Unique Features

**TikTok No Watermark:** DropZap fetches TikTok videos from the source CDN
before watermark application, resulting in completely clean downloads.

**Instagram Carousels:** DropZap packages all carousel slides into a ZIP file
automatically. Each slide is saved at its original uploaded resolution.

**Reddit Audio Merge:** Reddit stores video and audio as separate files.
DropZap automatically merges them into a single MP4 with complete audio.

**Facebook Albums:** Multi-photo Facebook albums download as ZIP files.

**YouTube Thumbnails:** All 5 thumbnail sizes available — Default (120×90),
Medium (320×180), High (480×360), Standard (640×480), Max-Res (1280×720).
